# Sleepy Sheep 🐑

AI 기반 수면 관리 및 감정 케어 웹 애플리케이션

Sleepy Sheep는 사용자의 건강한 수면 습관 형성을 돕기 위해 제작된 수면 웰니스 웹앱입니다.

사용자는 수면 기록, 꿈일기, Dreamy 상담, ASMR, 모닝콜, 코골이 감지 기능 등을 활용하여 자신의 수면 패턴을 관리할 수 있습니다.

## 주요 기능

### 🌙 수면 관리

- 수면 기록 작성
- 주간 및 월간 수면 통계
- 수면 규칙성 분석
- 수면 목표 관리
- 코골이 기록 기능

### 📖 꿈일기 및 감정 케어

- 꿈일기 작성
- Dreamy 고민 상담
- 감정 기록
- AI 기반 위로 메시지

### 🎵 수면 보조 기능

- ASMR 재생
- 모닝콜
- 코골이 감지
- 수면 메이트

### 🐑 캐릭터 콘텐츠

- 드리미 성장 시스템
- 쓰다듬기 · 먹이주기 · 양털깎기
- 양털 수집
- 갤러리 수집
- 방 꾸미기

### 🌎 다국어 지원

- 한국어
- 영어
- 일본어
- 중국어

### 📱 플랫폼 기능

- PWA 지원
- 홈 화면 바로가기 설치
- 브라우저 알림
- 백그라운드 모닝콜
